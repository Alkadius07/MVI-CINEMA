# 🎬 MVI Cinema

Website jadwal nonton bareng virtual untuk komunitas **Mobian Vtuber Indonesia (MVI)** di Discord.

## 🌐 Live Demo
👉 [alkadius07.github.io/MVI-CINEMA](https://alkadius07.github.io/MVI-CINEMA/)

## 📌 Tentang Proyek
MVI Cinema adalah event cinema virtual yang memungkinkan 
anggota dari discord komunitas Mobian Vtuber Indonesia untuk nonton film bareng secara gratis 
melalui Theater Room di Discord.

## ✨ Fitur
- Halaman utama dengan menampilkan branding komunitas
- Daftar jadwal film (Akan Datang & Sedang Tayang)
- Informasi film: genre, durasi, dan rating
- Navigasi sederhana (Home & Schedule)
- Tombol CTA untuk bergabung ke Discord server
- Desain responsif dengan tema dark blue

## 🛠️ Teknologi
- HTML
- CSS
- JavaScript (basic)
- GitHub Pages (Deployment)
